# M-Pesa Certificates Directory

This directory should contain the M-Pesa public certificate for security credential encryption.

## Required Files:
- ProductionCertificate.cer (for production)
- SandboxCertificate.cer (optional, for sandbox - usually not needed)

## How to obtain:
Download from Safaricom Daraja Portal:
- Production: https://developer.safaricom.co.ke/APIs/MpesaG2

## Note:
Certificate files are ignored by git (.gitignore) for security reasons.
